# AWS 자격증 공부 가이드

AWS Cloud Practitioner와 AI Practitioner 자격증 준비를 위한 체계적인 학습 자료입니다.

## 📚 포함된 자격증

### 1. AWS Certified Cloud Practitioner (CLF-C02)
- **난이도**: 기초
- **시험 시간**: 90분
- **문항 수**: 65문항
- **합격 점수**: 700/1000

AWS 클라우드의 기본 개념, 서비스, 보안, 아키텍처, 가격 및 지원에 대한 전반적인 이해를 검증합니다.

### 2. AWS Certified AI Practitioner (AIF-C01)
- **난이도**: 기초
- **시험 시간**: 120분
- **문항 수**: 85문항
- **합격 점수**: 700/1000

AWS의 AI/ML 서비스와 생성형 AI에 대한 기본 지식을 검증합니다.

## 🚀 사용 방법

### GitHub Pages로 호스팅하기

1. 이 저장소를 GitHub에 업로드합니다
2. 저장소 Settings > Pages로 이동
3. Source를 "Deploy from a branch" 선택
4. Branch를 "main" 선택하고 폴더를 "/ (root)" 선택
5. Save 클릭
6. 몇 분 후 `https://[username].github.io/[repository-name]/` 에서 접속 가능

### 로컬에서 실행하기

```bash
# 간단한 HTTP 서버 실행 (Python 3)
python -m http.server 8000

# 또는 Node.js 사용
npx http-server
```

브라우저에서 `http://localhost:8000` 접속

## 📖 학습 내용

### Cloud Practitioner
- ✅ 클라우드 개념 (24%)
- ✅ 보안 및 규정 준수 (30%)
- ✅ 클라우드 기술 및 서비스 (34%)
- ✅ 결제, 가격 및 지원 (12%)

### AI Practitioner
- ✅ AI 및 ML의 기초 (20%)
- ✅ 생성형 AI의 기초 (24%)
- ✅ AWS AI/ML 서비스 응용 (28%)
- ✅ 책임 있는 AI 지침 (14%)
- ✅ AI 솔루션의 보안, 규정 준수 및 거버넌스 (14%)

## 🎯 주요 특징

- **체계적인 구성**: 시험 도메인별로 정리된 내용
- **서비스 비교표**: 유사 서비스 간의 차이점 명확히 비교
- **핵심 키워드**: 시험에 자주 나오는 키워드 강조
- **시험 팁**: 실전 문제 패턴 및 답변 전략
- **깔끔한 UI**: 가독성 높은 디자인
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
